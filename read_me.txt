navbar : permet de créer une navbar.

navbar dark : permet de mettre tous les attributs de la navbar en mode sombre (texte, lien, ...).

bg-dark : permet de mettre le fond d'un élément en noir.

navbar-expend : permet d'aligner horizontalement les éléments de la navbar.

contenaire-fluid : élément qui prend 100% de la largeur.

navbar-brand : permet de définir l'élément qui servira de titre ou de logo sur la navbar.

navbar-nav : permet de définir les éléments de navigation de la navbar.

position : p pour padding ; x pour définir le padding droite et gauche ; t pour le haut ; b pour le bas.

row : permet d'initialiser les colonnes.

col-x : permet de définir une colonnes de taille x.

card : permet de créer une carte

card-body : permet de définir le centre de la carte, le corps, il contient le card-title et le card-text.
